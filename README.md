# Assignment 4: Character Animation Control
Use model and animation from Mixamo and blend the animation with multiple actions (Skeletal Animation)

## Overview
- use skeletal_animation.cpp as a template along with bone.h and animator.h
- import granny model and granny rig animation from Mixamo
- set the FSM for state handling (idle, walk, run, punch, kick)
- blend the animation by using the inter-state animation, e.g, idle -> idle_walk -> walk -> walk_run -> run -> run_walk -> walk_idle -> idle
