# Assignment 3: A Simple 3D Game
Create a 3D game with OpenGL. Contains 3D model loading, camera following, and object intersection

## Overview
- A cow on a grassland with rigid trees and collectibles
- player controls a cow with WASD and mouse movement (following camera)
- player can collect the collectible by walking into it (AABB XZ Collision Checking)
- player gets stuck when walking into a tree
